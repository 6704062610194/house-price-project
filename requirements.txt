# 🏠 House Price Prediction Project

## 📌 รายละเอียดโปรเจค
โปรเจคนี้เป็นการพัฒนาโมเดล Machine Learning และ Neural Network
เพื่อทำนายราคาบ้านจากข้อมูลคุณลักษณะต่าง ๆ

---

## 📊 Dataset
- ใช้ข้อมูล House Prices
- มีทั้งข้อมูลตัวเลขและ categorical
- มี missing values ต้องทำ Data Preparation

---

## ⚙️ Data Preparation
- เติม missing values ด้วยค่าเฉลี่ย
- แปลง categorical เป็นตัวเลข (One-hot encoding)
- ทำ Feature Scaling (StandardScaler)

---

## 🤖 โมเดลที่ใช้

### 1. Machine Learning (Ensemble)
- Random Forest
- Gradient Boosting
- XGBoost
- ใช้ Voting Regressor รวมผลลัพธ์

### 2. Neural Network
- Input Layer
- Hidden Layer (128, 64 neurons)
- Output Layer
- ใช้ ReLU + Adam optimizer

---

## 🌐 Web Application
พัฒนาโดยใช้ Streamlit

ประกอบด้วย:
- หน้าอธิบาย ML Explain
- หน้าอธิบาย NN Explain
- หน้า Test ML
- หน้า Test Neural Network

---

## 📈 การทำงาน
รับ input จากผู้ใช้ เช่น:
- Lot Area
- Overall Quality
- Year Built
- Living Area
- Garage Cars

แล้วนำไปทำนายราคาบ้าน
